# florida-mbb-2024-2025-season
##Project Overview
This project analyzes Florida's 2025 season using NCAA play-by-play data (hoopR). 

The analysis includes: 
- Full-season metrics
- National Championship game breakdown
- Offensive & Defensive Efficiency
- eFG%
- Turnover Rate
- Rebounding %
- Clutch performance
- Possession-based efficiency estimates
Tools Used:
- R
- hoopR
- tidyverse
- ggplot2

##Side Note
I started with the championship game first because I initially did not intend to compare the tournament games with the regular season. That is why the data is loaded and ordered from the championship game, and the analysis is taken further into the actual season.
